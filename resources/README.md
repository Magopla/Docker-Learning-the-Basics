## 🏷️ Tags & Versions Best Practices

| Tag | When to Use |
|-----|-------------|
| `:latest` | Default, avoid in production |
| `:1.2.3` | Specific version, recommended |
| `:-alpine` | Small image, production OK |
| `:-slim` | Smaller than full, good for prod |
| `:latest` | Never use in production |
