# External Resources

This page collects useful places to keep learning after you finish the core exercises.

## Best Places To Start

If you only open a few links, make them these:

| Resource | Why it is useful |
| --- | --- |
| [Docker Documentation](https://docs.docker.com/) | Official docs for everyday Docker usage |
| [Docker Desktop](https://www.docker.com/products/docker-desktop/) | Installation and product information |
| [Docker Hub](https://hub.docker.com/) | Search for public images |
| [Docker Compose file reference](https://docs.docker.com/compose/compose-file/) | Compose syntax and options |

## Suggested Next Topics

After this repository, a good learning order is:

1. Write a simple `Dockerfile`.
2. Build your own image.
3. Learn Compose more deeply.
4. Learn image optimization and tagging strategy.
5. Explore container orchestration.
